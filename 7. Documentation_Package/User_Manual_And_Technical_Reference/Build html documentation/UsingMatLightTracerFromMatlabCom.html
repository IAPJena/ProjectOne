Using MatLightTracer From Matlab Command


Even though MatLightTracer toolbox has its own GUI which could be used to setup and analyze optical systems, it sometimes becomes neccessary to use the functionality of the toolbox from the matlab command window or an other external toolbox.  This section of the user manual describes some important points which could be helpful when trying to use the MatLightTracer toolbox with out its GUI. This enables integration of the toolbox to other projects.
