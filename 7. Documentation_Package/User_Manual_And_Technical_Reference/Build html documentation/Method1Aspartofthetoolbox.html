Method 1: As part of the toolbox

In this method some the existing codes of the toolbox are modified and the new feature becomes integrated as part of the toolbox.

Advantages:

  • It allows making use of already defined uicontrols for the graphical user interface.
  • The user interface of the new feature becomes similar to all windows of the toolbox.
  • The feature can easily be reused as it becomes the part of the toolbox.


Disadvantages:

  • Requires editing the existing code which could be error prone to the whole toolbox.
  • It requires some understanding of the existing code of the toolbox.
