Examples of Extending the Toolbox

To extend the functionalities of the toolbox there are two possibilities.

  1. Adding feature which becomes the integral part of the toolbox
  2. Adding features which is not integrated to the toolbox but uses the toolbox to setup and analyze optical systems.

Each approach has advantages and disadvantages. The following sections discuss each method by giving an example of extending the toolbox.
