Toolbox Structures and Classes

The backbones of the MatLightTracer toolbox are the structures and classes defined for different entities in the toolbox. All classes and struct defintions are place inside a folder whose name is the same as that of the class/struct. All functions for manupilating the struct/or objects of the class are placed in the same folder. Following is the list of structs and classes with brief description.
