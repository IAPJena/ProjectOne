Using MatLightTracer GUI


The MatLightTracer toolbox has its own GUI which could be used to setup and analyze optical systems. This section of the user manual describes basic steps to use the user interfaces of the toolbox. As describe in the previous section all functionalities of the toolbox can also be accessed with out using the GUI. That enables integration of the toolbox to other projects and is discussed in the last section section of the user manual.
