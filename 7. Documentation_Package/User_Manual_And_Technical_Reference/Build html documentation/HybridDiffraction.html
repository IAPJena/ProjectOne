Hybrid Diffraction


To account for diffraction in system aperture a hybrid diffraction model is implemented in the toolbox. That is ray is traced until exit pupil and then the complex field is reconstructed at the exit pupil. The complex field is then propagated from the exit pupil to the focal plane using simple diffraction algorithm (Far field diffraction integral).
